## Task 3 ‚Äì HR Attrition Dashboard (Power BI)

### Objective
This project is part of my internship **Task 3**, where the objective was to design and publish an **interactive Power BI dashboard** that highlights key HR attrition insights using **business storytelling and DAX measures**.

The dashboard focuses on:
- Understanding employee attrition patterns  
- Identifying high-risk employee groups  
- Comparing salary trends between attrited and retained employees  

It demonstrates **practical Power BI skills** commonly expected in **Business Analyst / Data Analyst interviews**.

---

### Key Features
- KPI Cards: **Total Employees, Total Attrition, Attrition Rate**
- Department-wise, Age-wise, and Gender-wise attrition analysis
- Salary comparison between **Attrition vs Retention**
- Interactive slicers and report-level tooltips
- Business insight storytelling using **DAX measures**

---

### Business Insights Derived
- Employee attrition is highest during early tenure years  
- Certain departments experience significantly higher attrition  
- Employees with lower average salaries show a higher likelihood of attrition  
- Attrition trends vary across age groups and education fields
